# 🧠 ZEUS Terminal Hack Game

A terminal-based hacking simulation game built with Python and Textual.

You are trapped inside the ZEUS system.  
Solve puzzles, decode messages, explore files, and uncover the hidden truth behind the system.

---

## 🚀 Features

- 🎮 Multi-stage mission system
- 💻 Terminal-style interactive interface
- 🧩 Logic & math puzzles
- 🗂️ Virtual file system (ls / open)
- 🔐 Login system with hidden password discovery
- 💡 Progressive hint system
- 🧠 Memory fragments system
- 🔀 Multiple endings:
  - Escape the system
  - Trust the AI
  - Destroy everything

---

## 🕹️ How to Play

Type commands into the terminal input.

### 📌 Commands:
- `help` → Show available commands  
- `ls` → List files (mission-based)  
- `open <file>` → Open a file  
- `login <password>` → Login to system  
- `decode <key>` → Decode message  
- `solve <answer>` → Solve puzzles  
- `hint` → Get hints  
- `memory` → Access memory fragments  
- `trace` → Track system  
- `exit` → Exit the game  

---

## 🧠 Story

You are inside a system called **ZEUS**.

At first, it looks like a normal simulation.  
But slowly, you discover:

- Hidden files
- Corrupted messages
- AI warnings
- Memory fragments about YOU

The system is not just a game...  
It is trying to tell you something.

---

## 🎯 Missions

### Mission 1
Decode encrypted message and unlock first access.

### Mission 2
Explore system files and find admin password.

### Mission 3
Solve mathematical pattern challenge.

### Mission 4
Discover hidden encrypted file (`secret.enc`).

### Mission 5
Recover memory fragments and unlock "truth".

### Mission 6
Make the final choice:
- Escape
- Trust AI
- Destroy system
